## Dica importante
Após criar o formulário:
1. Clique em **Enviar** (botão roxo no topo)
2. Copie o link curto (ex: https://forms.gle/XXXXX)
3. Cole no arquivo `js/config.js`:
   ```js
   forms: {
     r32: { url: "https://forms.gle/XXXXX", deadline: "2026-06-27T23:59:00" },
   ```
4. Salve, faça `git add js/config.js && git commit -m "Adiciona link forms R32" && git push`
